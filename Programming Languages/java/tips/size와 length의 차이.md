|       | `size()`  | `length` | `length()` |
| ----- | --------- | -------- | ---------- |
| 적용 대상 | 컬렉션 프레임워크 | 배열       | 문자열        |
| 타입    | 메서드       | 속성       | 메서드        |
## `size()`
`ArrayList`, `LinkedList`, `HashSet`, `HashMap` 같은 컬렉션 프레임워크의 크기를 반환할 때 사용한다.
- 내부적으로 저장된 요소의 개수를 반환한다.
## `length`
배열(`Array`)의 길이를 반환할 때 사용한다.
## `length()`
문자열은 객체이므로 `length()` 메서드로 길이를 가져온다.
