# Python Set
```py
set_1 = set([0, 4, 2, 0])
print(set_1)    # {0, 2, 4}

set_2 = set('esens')
print(set_2)    # {'n', 'e', 's'}
```
* 중복 허용 X
* 순서 X

## 집합 연산
* 교집합(`intersection`) - `&`
* 합집합(`union`) - `|`
* 차집합(`difference`) - `-`
